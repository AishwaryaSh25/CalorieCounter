About Calorie Counter AI Agent

Your intelligent nutrition companion powered by AI

Project Overview

The Calorie Counter AI Agent is a revolutionary web application that analyzes ANY food using Google Gemini AI to provide personalized recommendations based on your health profile.

Enter any food name - from "homemade pasta" to "street tacos" to "grandma's apple pie" - and get instant AI-powered analysis considering your weight, height, age, activity level, and health conditions for tailored nutritional guidance.

Key Features

  • Personalized Profiles: Create detailed health profiles with physical metrics and health conditions
  • Google Gemini AI: Advanced AI analysis for unlimited food variety
  • Unlimited Food Analysis: No restrictions - analyze any food you can think of
  • Portion Recommendations: Optimal serving sizes based on your caloric needs
  • Health Condition Awareness: Special considerations for diabetes, hypertension, and more
  • No Database Required: Pure in-memory storage, no setup complexity

Technology Stack

Backend Technologies
  • Java 17: Modern programming language
  • Spring Boot 3.2: Application framework
  • Google Gemini: AI analysis engine
  • In-Memory Storage: HashMap-based data storage
  • Maven: Build and dependency management
Frontend Technologies
  • Bootstrap 5: Modern CSS framework
  • Font Awesome: Professional icons
  • HTML5/CSS3: Modern web standards
  • JavaScript: Interactive user experience
  • Responsive Design: Mobile-first approach

How It Works

1. Profile Creation

Create your health profile with pre-filled defaults for quick testing, including metrics, activity levels, and health conditions.

2. Any Food Input

Enter ANY food name - from "homemade pasta" to "street tacos" - no database limitations or restrictions.

3. Gemini AI Analysis

Google Gemini AI analyzes the food against your health profile and provides personalized recommendations instantly.

AI Reasoning Engine

Our AI reasoning engine considers multiple factors when analyzing foods:

Health Factors
  • Existing health conditions
  • Dietary restrictions
  • Allergen considerations
  • Age and gender-specific needs
Nutritional Factors
  • Caloric density and daily needs
  • Macronutrient balance
  • Micronutrient content
  • Food category and processing level
